# Movies CRUD App 🎬

A full-stack movie management application built with Next.js, Firebase, and Firestore.

This project is focused on learning firestore concepts including:

- Authentication (signup, login and logout via firebase itself)
- Firestore database CRUD operations (for movies)
- Protected routes (only allow access to logged in users)
- Component-based UI with shadcn/ui (As we use it in AgentsOnly)


## ✨ Features

### Authentication
- User Signup
- User Login
- Logout functionality
- Firebase Authentication integration
- Protected movie routes

### Movies CRUD
Users can:

- ➕ Add movies
- 📖 View movies
- ✏️ Edit movie details
- ❌ Delete movies

### UI
- Responsive design
- Clean modern interface using shadcn/ui
